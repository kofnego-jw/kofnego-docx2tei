Help

Please don't panic, if the application is not working.

View or Download

Using "View" will probably cause your browser to block a popup, if you allow the popup, the result of the conversion will be displayed in the browser. Using "Download" will cause your browser to download and save a file.

Browser Support

Please note that you must use a modern Web Browser (capable of doing stuffs with the HTML 5 Files API). These are:

  • Firefox, from Version 13
  • Chrome from version 20
  • Internet Explorer from version 10
  • Safari from version 6
  • Opera from version 12.1

If your browser is too old, sorry...

Your browser must enable Javascript.
