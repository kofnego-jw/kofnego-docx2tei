Impressum

The development of this web application was funded by the Commission for Modern Austrian History.

It uses the following open source software libraries:

  • Stylesheets of the TEI Consortium
  • Spring Framework (Boot, MVC)
  • Saxon XSLT Processor
  • Angular JS
  • FileSaver.js
  • XStream XML Object Mapper
  • Jackson JSON Object Mapper
